#Yelp Crawler

Python scripts to crawl and extract all reviews for restaurants from yelp.
1. Install Required Python Packages
urllib2
BeautifulSoup
pyquery 
lxml
datetime
json
selenium
yelpapi
json
fuzzywuzzy

2. Get a list of restaurant names for which you need to get reviews. Run get_yelp_Data.py script with restaurant names files as input.
Modify script accordingly.

3. Give output from 2nd step as input to getReviewsFromYelp.py.
